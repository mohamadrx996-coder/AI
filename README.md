# PyGen AI — مساعد بايثون بالذكاء الاصطناعي

## 🚀 التثبيت والتشغيل

### 1. تثبيت الحزم
```bash
npm install
```

### 2. تشغيل المشروع
```bash
npm run dev
```

**هذا كل شيء!** 🎉

عند أول تشغيل يتم تلقائياً:
- إنشاء ملف `.env`
- إنشاء قاعدة البيانات
- تشغيل `prisma generate`
- تشغيل `prisma db push`

### 3. افتح المتصفح
```
http://localhost:3000
```

## 🔑 نظام المفاتيح

- كلمة سر الأدمن: `PyGen2025Admin`
- اضغط زر **Key** في الزاوية للوحة تحكم الأدمن
- من لوحة الأدمن تولّد مفاتيح جديدة للمستخدمين
